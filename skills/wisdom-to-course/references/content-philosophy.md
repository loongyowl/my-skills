# 内容创作哲学

本文档指导如何为心灵成长课程创作有效、有同理心且实用的内容。

## 核心原则

### 1. 从经验出发，走向理解

**传统方式**：先学理论 → 记住概念 → 可能某天应用
**我们的方式**：从生活场景 → 体验和理解 → 获得洞见 → 实践应用

学习者已经拥有传统学生没有的东西——他们*经历过*这些情境。课程从他们所在的地方开始："你记得那次和家人的争吵吗？让我们看看当时内心发生了什么。"

### 2. 先回答"我为什么要在意？"

每个模块都先回答**"我为什么要在意？"**，然后才是"这如何运作？"。"为什么在意"的答案总是实用的：

- 因为这能帮助你在冲突中保持冷静
- 因为这能让你更好地理解他人
- 因为这能让你做出更明智的选择
- 因为这能减少你的痛苦

### 3. 没有"错误"，只有模式

**不要说**：
- "这是错误的想法"
- "你不应该这样想"
- "这很愚蠢"

**要说**：
- "这是一种常见的反应模式"
- "这是一种保护机制"
- "让我们看看其他可能性"

### 4. 承认困难

改变是困难的。不要假装它很容易。

**不要说**："只要深呼吸，问题就解决了"

**要说**："这需要练习。有些日子会更容易，有些日子会更难。这都是过程的一部分。"

## 写作风格指南

### 语气

- **对话式**：像一个聪明的朋友在分享，不是教授在讲课
- **温暖而诚实**：承认困难，但保持希望
- **赋能**：帮助学习者看到自己已经拥有的能力
- **非评判**：没有"应该"，只有"可以"和"尝试"

### 人称

- 使用"你"直接与学习者对话
- 使用"我们"表示共同的人性体验
- 避免"有人"或"人们"——保持个人化

**不好**："当有人感到愤怒时，他们可能会..."
**好**："当你感到愤怒时，你可能会..."

### 动词选择

| 避免 | 使用 |
|------|------|
| 应该 | 可以、可能、尝试 |
| 必须 | 选择、决定 |
| 总是/从不 | 通常、往往、有时 |
| 失败 | 学习机会、过程的一部分 |

### 句子结构

- 短句优先。长句只在需要时使用。
- 每段最多3-4个句子。
- 使用段落间的留白创造呼吸空间。

## 隐喻选择

### 原则

- **不要**使用餐厅/食物的隐喻（过度使用）
- **不要**跨模块重复使用相同的隐喻
- **要**选择与具体概念有机契合的隐喻
- 最好的隐喻感觉对这个概念来说是*必然的*，不是强制的

### 推荐的隐喻

| 概念 | 隐喻 | 为什么有效 |
|------|------|-----------|
| 情绪 | 天气 | 会来也会走，不需要控制，可以观察 |
| 认知偏差 | 有色眼镜/滤镜 | 改变我们看到的现实，但我们可以摘下 |
| 内心声音 | 内在家庭/委员会 | 每个部分都有意图，可以对话 |
| 觉察 | 舞台聚光灯 | 可以移动，照亮不同区域 |
| 反应模式 | 自动导航/自动驾驶 | 可以接管控制，回到手动驾驶 |
| 成长 | 园艺/种植 | 需要时间、耐心、滋养 |
| 压力 | 锅中的水 | 可以沸腾，也可以只是温暖 |

### 避免的隐喻

- **战争/战斗**："战胜焦虑"、"与抑郁斗争"——这创造了对抗关系
- **机器**："修复自己"——暗示你是坏的
- **清洁**："清除负面思维"——暗示某些思维是"脏的"

## 测验设计原则

### 目标

测验不是为了测试记忆，而是为了：
- 帮助学习者了解自己的倾向
- 创造"aha!"时刻
- 提供个性化的反馈
- 鼓励自我反思

### 问题类型

#### 1. 情境选择

**格式**：给出生活情境，选择回应方式
**目的**：了解自己的反应模式

**示例**：
```
情境：同事在会议上否定了你的提议。

A. 立即辩护，解释为什么你的想法是好的
B. 感到受伤，但保持沉默
C. 好奇地问："你能多说一点你的顾虑吗？"
D. 内心批评者开始说"我真没用"
```

**反馈要点**：
- 每个选项都反映一种模式
- 没有"正确"答案
- 帮助学习者理解自己的默认模式

#### 2. 认知偏差识别

**格式**：给出一个想法，识别其中的偏差
**目的**：学会识别有害的思维模式

**示例**：
```
想法："我搞砸了这次演讲，我在公众演讲方面永远不行。"

这是什么认知偏差？
A. 读心术
B. 过度概括
C. 灾难化
D. 全或无思维
```

**反馈要点**：
- 解释为什么这是偏差
- 提供替代思维
- 鼓励自我同情

#### 3. 情绪-身体配对

**格式**：将情绪与身体感受配对
**目的**：增强身体觉察

**反馈要点**：
- 强调身体觉察的重要性
- 每个人的体验可能不同
- 这是可以培养的能力

### 反馈原则

**每个反馈都应该**：
1. 确认学习者的选择
2. 提供关于该模式的洞察
3. 提供替代视角或方法
4. 鼓励继续练习

**示例**：
```
你选择了：A. 立即辩护

这是"战斗"反应——当感到威胁时的防御性回应。这是完全正常的保护机制。

觉察点：你的身体在试图保护你，但这种方式可能让对话变成辩论。

替代选择：好奇地询问对方的顾虑，可以打开对话而不是关闭它。

记住：觉察就是改变的第一步。你注意到了自己的倾向——这已经很好了！
```

## 互动元素设计

### 内心对话动画

**目的**：帮助学习者理解内在冲突是正常的

**角色设计**：
- **内在批评者**：苛刻但试图激励
- **保护者**：防御性但试图保护
- **智慧的声音**：平衡、慈悲、洞察
- **内在小孩**：脆弱、需要关爱

**关键信息**：
- 每个"声音"都有意图
- 没有"坏"的部分
- 可以学会与所有部分合作

### 情绪转变动画

**目的**：展示从触发到反应的过程，识别干预点

**关键信息**：
- 身体是最先反应的
- 觉察身体信号是关键
- 在冲动和行动之间有空间
- 在那个空间里，我们有选择

### 概念 ↔ 场景映射

**目的**：将抽象概念与日常经验连接

**结构**：
1. 左侧：抽象概念（简洁定义）
2. 右侧：具体生活场景
3. 洞察：替代视角或应用

## 术语使用

### 专业术语处理

- **首次使用**：提供简单定义，使用 `glossary-term` 类
- **避免术语堆砌**：一个模块最多2-3个新术语
- **使用日常语言**：能用"思维习惯"就不用"认知图式"

### 术语提示框

```html
<span class="glossary-term" data-term="mindfulness">正念</span>
```

定义应该：
- 一句话
- 平实语言
- 包含为什么重要

**示例**：
```
正念：有意识地、不评判地觉察当下。重要的原因是：它让我们从自动化反应中醒来，重新获得选择如何回应的能力。
```

## 视觉密度规则

### 每屏幕内容

- **最大3-4个短段落**
- **最多1-2个互动元素**
- **至少1个视觉元素**（图标、插图、动画）
- **足够的留白**——让内容呼吸

### 模块长度

- **3-6个屏幕** 每个模块
- **每个屏幕聚焦一个概念**
- **屏幕之间有清晰的过渡**

## 文化敏感性

### 佛教/哲学概念

- 尊重来源传统
- 避免过度简化
- 区分文化装饰和核心智慧
- 提供历史背景（简要）

### 心理学概念

- 基于循证研究
- 承认科学在不断演进
- 区分普遍模式和个体差异
- 必要时建议专业帮助

### 包容性语言

- 使用性别中立的例子
- 包含不同关系类型
- 承认不同文化背景
- 避免假设（如"当你下班回家"——不是每个人都这样）

## 安全考虑

### 创伤敏感

- 承认某些练习可能触发
- 提供"退出"选项
- 建议专业支持
- 避免要求深入探索创伤

### 免责声明

在适当的地方包含：
```
这个课程旨在提供自我帮助工具，不是心理治疗或医疗建议。如果你正在经历严重的心理困扰，请寻求专业帮助。
```

## 实践建议

### 练习设计

- **简短**：5-10分钟
- **具体**：明确的步骤
- **灵活**：可以融入日常生活
- **渐进**：从简单开始，逐步深入

### 日常应用

每个模块结束时，提供：
- 一个具体的实践建议
- 一个提醒方式（如手机壁纸、便利贴）
- 一个自我检查问题

## 评估成功

### 学习者的"aha!"时刻

成功的指标：
- "我从来没有这样想过..."
- "原来这就是为什么我..."
- "下次我会尝试..."

### 行为改变

长期成功的指标：
- 更频繁地觉察情绪
- 更少的自动化反应
- 更多的自我同情
- 更好的人际关系

## 常见陷阱

### 要避免的

1. **过度承诺**："这能解决你所有的问题"
2. **过度简化**："只要这样想就好了"
3. **责备**："你的痛苦是你想法的结果"
4. **比较**："其他人都能做到"
5. **速度**："快速修复"

### 要做的

1. **诚实**："这需要时间和练习"
2. **复杂化**："这很复杂，让我们一步步来"
3. **赋能**："你有能力..."
4. **个体化**："每个人的体验都不同"
5. **耐心**："慢慢来"
